# BlogApp

A full-featured blog web application built with Spring Boot. This project allows users to create, read, update, and delete blog posts, providing a simple platform for personal or group blogging.

## Features

- User authentication and authorization (Login/Register)
- Create, edit, and delete blog posts
- View all posts or individual post details
- Comment system for posts
- Responsive design (works on desktop and mobile)

## Tech Stack

- **Backend:** Spring Boot (Java), Spring MVC, Spring Data JPA, Spring Security
- **Frontend:** Thymeleaf, HTML, CSS, Bootstrap
- **Database:** MySQL / PostgreSQL (configurable)
- **Build Tool:** Maven
